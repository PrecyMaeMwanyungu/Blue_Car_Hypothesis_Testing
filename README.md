# Blue_Car_Hypothesis_Testing
This is the data repository for a project by a student of Moringa School to test the hypothesis if there is a difference between the number of blue cars taken on weekdays from postal code 75011 and postal code 75015 of Autolib Electric car sharing company.Autolib is operated by the Bollor√© Group enterprise, which won the contract to develop the service and to supply Paris with electric cars and stations.
